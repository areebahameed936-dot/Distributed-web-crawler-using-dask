🕷️ Distributed Web Crawler using Dask

A high-performance distributed web crawler that fetches web pages in parallel using Dask, performs text analysis, sentiment detection, and generates interactive dashboards.

 🚀 Features
- Distributed crawling using Dask (4 workers)
- Word frequency analysis
- Sentiment analysis (TextBlob)
- Trend detection
- Source ranking
- Interactive dashboard

🛠 Tech Stack
- Python
- Dask
- BeautifulSoup
- Pandas
- TextBlob
- Matplotlib
